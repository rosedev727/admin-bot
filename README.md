A lightweight, plug-and-play **Discord administration bot** written in **Python 3** with  
[`discord.py` v2](https://github.com/Rapptz/discord.py).  
Built by **Rose Development Team** for painless server moderation and ticket handling.

---

## ✨ Features

| Category      | Command / Feature                 | Description                                             |
| ------------- | --------------------------------- | ------------------------------------------------------- |
| Moderation    | `/warn`, `/unwarn`, `/warn_list`  | Add, remove, or list user warnings (stored locally).    |
|               | `/ban`, `/unban`, `/kick`         | Standard member discipline tools.                       |
|               | `/mute`, `/unmute`                | Role–based mute with optional auto-unmute timer.        |
|               | `/purge`                          | Bulk-delete up to 100 recent messages.                  |
| Utilities     | `/say`                            | Make the bot speak for you (staff-only).                |
| Tickets       | `/ticket <subject>`               | Creates a private support channel for the user.         |
|               | `!close`                          | Staff command to close (delete) the ticket channel.     |
| Logging       | Mod-log channel                   | Every action is mirrored to a channel of your choice.   |
